# Simple Calculator iOS App

## Overview
This is a simple iOS application that demonstrates basic calculator functionality by allowing users to add two numbers. The app is built using **UIKit** and **Storyboards** and is intended for learning and practice purposes.

---

## Features
- Input two numbers using text fields
- Add numbers with a single button tap
- Display the result instantly
- Simple and beginner-friendly UI

---

## UI Components
- UITextField for first number input
- UITextField for second number input
- UIButton to perform addition
- UILabel to display the result

---

## Tech Stack
- **Language:** Swift
- **UI Framework:** UIKit
- **Layout:** Storyboards
- **Platform:** iOS


## Requirements
- Xcode 14+
- iOS 15+ (recommended)
- macOS compatible with Xcode version
